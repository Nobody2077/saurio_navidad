# SaurioNavidad

App Flutter para Android: un arbol de Navidad emocional que crece con recuerdos, una mascota llamada Saurio y un cofre para capsulas especiales.

## Estado actual

- Proyecto base Android listo para abrir en Visual Studio Code.
- Pantalla principal disenada.
- Arbol con profundidad visual, luces y crecimiento segun recuerdos.
- Mascota Saurio con ambiente segun la hora.
- Cofre bajo el arbol con capsulas bloqueadas.
- Calendario de diciembre: desbloquea dias y muestra recuerdos cuyo dia del mes ya llego.
- Formulario visual para agregar recuerdos de nota, foto, audio, lugar, meta o momento.

## Comandos utiles

```powershell
flutter pub get
flutter run
flutter build apk --release
```

El nombre visible de la APK en Android es `SaurioNavidad`.
